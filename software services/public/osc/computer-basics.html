1 What is a Computer?

A computer is an electronic device that processes data and performs tasks based on instructions (programs).

Main Parts:

  • CPU (Processor) - The "brain" that does calculations
  • RAM (Memory) - Temporary storage for running programs
  • Hard Drive/SSD - Permanent storage for files
  • Monitor - Displays what you see
  • Keyboard & Mouse - Input devices to control the computer
üí° Think of RAM as a desk (temporary workspace) and Hard Drive as a filing cabinet (permanent storage).
2 Turning On/Off Your Computer

To Turn ON:

  • Press the power button (usually has ‚èª symbol)
  • Wait for Windows to load (1-2 minutes)
  • Enter your password if required

To Turn OFF (Properly):

  • Click the Start button (Windows icon, bottom-left)
  • Click Power icon
  • Select Shut down

3 Using the Desktop & Start Menu

Desktop Elements:

  • Icons - Shortcuts to programs and files
  • Taskbar - Bottom bar with Start button and open programs
  • System Tray - Right side with clock, WiFi, volume

Start Menu:

  • Click Windows icon or press Windows key
  • Type to search for any program
  • Pin favorite apps for quick access
üí° Press Windows key + D to quickly show/hide desktop!
4 Keyboard Shortcuts

Essential Shortcuts:

  • Ctrl + C - Copy selected item
  • Ctrl + V - Paste copied item
  • Ctrl + X - Cut (move) selected item
  • Ctrl + Z - Undo last action
  • Ctrl + S - Save current file
  • Ctrl + A - Select all
  • Alt + Tab - Switch between open windows
  • Alt + F4 - Close current window
üí° Learning shortcuts saves hours of time! Start with Ctrl+C and Ctrl+V.
5 Mouse Operations

Mouse Actions:

  • Single Click - Select an item
  • Double Click - Open a file or program
  • Right Click - Show options menu
  • Click & Drag - Move items or select text
  • Scroll Wheel - Scroll up/down on pages
üí° Right-click is your friend! It shows all available options for any item.
6 Files & Folders

Understanding Files:

  • File - A document, image, video, or program
  • Folder - A container to organize files
  • File Extension - Shows file type (.docx, .pdf, .jpg)

File Explorer (Windows + E):

  • Create folder - Right-click ‚Üí New ‚Üí Folder
  • Rename - Right-click ‚Üí Rename (or F2)
  • Delete - Select and press Delete key
  • Copy/Move - Ctrl+C/X then Ctrl+V
üí° Keep files organized in folders like Documents, Pictures, Downloads!
7 Internet & Web Browsers

Web Browsers:

  • Chrome - Most popular, by Google
  • Edge - Built into Windows
  • Firefox - Privacy-focused

Browser Basics:

  • Address Bar - Type website URLs here
  • Tabs - Open multiple pages (Ctrl+T for new tab)
  • Bookmarks - Save favorite sites (Ctrl+D)
  • Back/Forward - Navigate between pages
üí° Use Google (google.com) to search for anything - just type your question!
8 Email Basics

Email Address Format:

yourname@gmail.com (name + @ + provider)

Common Actions:

  • Compose - Write new email
  • Reply - Respond to sender
  • Forward - Send to someone else
  • Attach - Add files to email

Email Tips:

  • Use clear subject lines
  • Be polite and professional
  • Check spelling before sending
9 Computer Security

Stay Safe Online:

  • Strong Passwords - Use 8+ characters with numbers & symbols
  • Don't share passwords - Ever!
  • Avoid suspicious links - Don't click unknown emails
  • Keep Windows updated - Updates fix security holes

Warning Signs of Scams:

  • "You won a prize!" - Probably fake
  • Urgent requests for money or info
  • Emails with spelling mistakes
  • Unknown senders asking for passwords
‚ö†Ô∏è Banks and companies NEVER ask for passwords via email!
10 Keeping Your Computer Healthy

Maintenance Tips:

  • Restart regularly - Fixes many issues
  • Delete unused files - Free up space
  • Empty Recycle Bin - Right-click ‚Üí Empty
  • Install updates - Settings ‚Üí Update & Security
  • Use antivirus - Windows Defender is built-in
üí° Restart your computer at least once a week for best performance!
